# Проект: "Лендинг для Дениса" 

## Описание: 

Одностраничный сайт портфолио для дизайнера с адаптивом и бургер меню.

## Функционал: 
 
* Бургер меню
* Адаптивная верстка

## Используемые технологии: 

* Flat БЭМ
* Git 
* HTML
* SCSS
* JS (ES6)
* WebPack
* Normalize.css
